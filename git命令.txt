git 提交


第一步 
$ git status   
 查看更新了哪些文件



第二步 
$ git add -A   
表示新增修改的文件到缓存列表


第三步
$ git commit -m "备注说明"
这个命令表示添加备注

第四步 
$ git push -u origin master:master   
提交到Git仓库。这里master为我自己的分支的名称，实际应用中，你要改成自己的分支的名称



git  更新


$ git pull origin master   //更新
